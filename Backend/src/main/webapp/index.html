CA3 - REST Backend

This is a backend for this matching React Front End

This Backend provides, out of the box:

  • Token Based Authentication
  • Two predefined user roles (with a sample REST-endpoint for each)
    • User
    • Admin
  • It user an in-memory database, to let you run the server right after cloning.
    You must change this to point to your own MySQL instance, either locally or remote
  • Built in filters to allow CORS
  • It stores passwords in PLAIN-TEXT, you must FIX THAT
